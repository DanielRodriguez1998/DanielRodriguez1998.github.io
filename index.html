Sobre mí

Licenciado en biotecnología genómica con maestría en inmunobiología, especializado en análisis e interpretación de datos biológicos, machine learning y estadística aplicada a datos biomédicos. Apasionado por construir soluciones prácticas para problemas de salud y ciencia, combinando R y Python en pipelines eficientes.

Proyectos destacados

Bioestadística

Repositorio con análisis estadísticos aplicados a ciencias biológicas, incluyendo ANOVA, regresión, correlaciones, análisis multivariados y visualizaciones en R.

Ver código en GitHub

Tesis de cáncer de mama (transcriptómica)

Pipeline completo para analizar datos transcriptómicos de cáncer de mama: deconvolución inmunológica, reducción dimensional (UMAP, PCA), clustering (PAM, HDBSCAN), análisis de supervivencia (modelos de Cox), índices pronósticos y redes génicas.

Ver código en GitHub

Predictor de Mechanism of Action (MoA)

Modelo en Python basado en TabTransformer para predecir mecanismos de acción de compuestos usando datos transcriptómicos. Implementa label smoothing, early stopping, scheduler y validación cruzada multisemilla.

Ver código en GitHub
